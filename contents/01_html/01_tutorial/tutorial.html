ルール1 タグを使って文章を書く

まずはシンプルに考えるために赤字部分だけを確認します。

タグルールの確認

<!DOCTYPE html>
  <html lang="ja" dir="ltr">
    <head>
      <meta charset="utf-8">
      <title>10分で学ぶHTML ゼロからスタート</title>
    </head>
    <body>
      <p>10分で学ぶHTML ゼロからスタート</p>
    </body>
  </html>

この部分だけ見てみると、<p>と</p> の間に"10分で学ぶHTML ゼロからスタート"というコメントの記載があります。 この"<"と">"で囲まれた部分がタグです。 このサンプルの場合、 <p>のほかに<title>、<head>、<body>などもタグです。 HTMLにはこのタグを使って文章を書くというルールがあります。

なお、今回ピックアップした<p>はparagraph(段落、節)のpです。文章を書くときに使うタグです。

paragraphタグの使用例


  　<p>10分で学ぶHTML ゼロからスタート</p>

タグは開始タグと終了タグの2種類あります。 paragraphタグの場合、開始タグが<p>で、 終了タグがバックスラッシュ" / "が付いた</p>というルールがあります。 文章はこの開始タグ<p>と終了タグ</p>の間に書く必要があります。

他のタグも同様に、例えば、ウェブページタイトルはtitleの開始タグ<title>と終了タグ</title>の間に、 HTMLヘッダと呼ばれているものはheadの開始タグ<head>と終了タグ</head>の間に書くといった 開始タグと終了タグの間に記述するというルールがあります。覚えておいてください。

ルール2 headとbodyを使ってHTMLを構成する

HTMLではheadタグで囲まれたヘッダとbodyタグで囲まれたボディをHTMLファイル内に用意する必要があります。

まずはhead(ヘッダ)を確認してみます。

HEADの確認

<!DOCTYPE html>
  <html lang="ja" dir="ltr">
    <head>
      <meta charset="utf-8">
      <title>10分で学ぶHTML ゼロからスタート</title>
  </head>
    <body>
      <p>10分で学ぶHTML ゼロからスタート</p>
    </body>
  </html>

headにはウェブページの情報を記述します。 開始タグは<head>、終了タグは</head>です。 その間に<meta charset="utf-8">や<title>xxx</title>がありますが、 これがheadに記載されたウェブページの情報となります。

<meta charset="utf-8">は文字形式(キャラクターセット)を示す情報ですが、まずはこのまま記載しておいてください。 プログラミングに親しい人ならよくご存じだと思いますし、そうでない人は深入りする必要はありません。

<title>xxx</title>はその名の通りタイトルです。ウェブページのタイトルを記載しておきます。 ただしここの情報はディスプレイには表示されません。Googleなどで検索したときに検索ブラウザ上で表示されるタイトルになります。 (ウェブブラウザのタブには表示されているかもしれません。)

つぎにbody(ボディ)を確認してみます。

BODYの確認

<!DOCTYPE html>
  <html lang="ja" dir="ltr">
    <head>
      <meta charset="utf-8">
      <title>10分で学ぶHTML ゼロからスタート</title>
    </head>
    <body>
      <p>10分で学ぶHTML ゼロからスタート</p>
  </body>
  </html>

bodyにはウェブページ本文を記述します。 開始タグは<body>、終了タグは</body>です。 その間に<p>10分で学ぶHTML ゼロからスタート</p>がありますが、 これがbodyに記載されたウェブページ本文の情報となります。

今回は<p>タグがひとつですが、必要に応じて複数記載することができます。 タイトルタグと異なり、これはウェブページ上に記載される情報となります。

HTML Styleの3つの設定方法

設定方法は3種類あります。タグに直接記載するインライン設定、ヘッダ内に記載するインターナル設定、そして外部ファイルを参照する外部設定。 以下にてそれぞれ紹介します。

[a]インライン設定

インライン設定はタグの中に直接Style情報を記載します。本文中の特定箇所だけ個別に設定できるので便利です。 ただ、使いすぎると保守性が低下するので、インターナル設定や外部設定と併用することをお勧めします。

インライン設定の例

<p style="color: red;">インライン設定の例</p>

[b]インターナル設定

インターナル設定はヘッダの中にStyle情報を記載します。タグごとや、特定のID、Classと呼ばれる単位に対して まとめて設定することができます。 以下の例では全ての<p>要素に対して文字が赤くなるような設定をしています。 ウェブページごとにまとめて設定したい場合に使用してみてください。

インターナル設定の例

<!DOCTYPE html>
  <html lang="ja" dir="ltr">
    <head>
      <style>
        p {color: red;}
    </style>
    </head>
    <body>
      <p>インターナル設定の例</p>
    </body>
  </html>

[c]外部設定

外部設定は外部のCascading Style Sheets(CSS)と呼ばれるファイルにStyle情報を記載します。 Style情報をウェブページに対してまとめて設定するときに便利です。 ただし、Style情報の変更が参照している全てのファイルに影響ができることに注意が必要です。

外部設定の例(HTMLファイル)

<!DOCTYPE html>
  <html lang="ja" dir="ltr">
    <head>
      <link rel="stylesheet" href="./style.css">
    </head>
    <body>
      <p>外部設定の例</p>
    </body>
  </html>

外部ファイルを参照するときは、下記のようにCSSファイルを引き当てる必要があります。

外部ファイルの参照方法

<link rel="stylesheet" href="./style.css">

外部ファイルの記載方法はインターナル設定の記述と同じです。ファイルの拡張子は".css"とする必要があります。

外部設定の例(外部ファイル/style.css)

p {color: red;}
